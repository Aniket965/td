# react-clean-drawer
> Simple and clean Drawer Component for React.js
> [NPM](https://www.npmjs.com/package/tdz)
### Installing

```sh
npm install tdz -g
```

### Usage
 - initializing
 ```sh
 tdz init
 ```
 - adding a task
 ```sh
 tdz add "task disp"
 ```
 -  removing task with id
 ```sh
 tdz rm 2
 ```
 - listing all tasks
 ```sh
 tdz
 ```
- removing all taks
 ```sh
 tdz rm -a
 ```
